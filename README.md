# Syntecxhub_Restaurant_Billing_System

**Syntecxhub Internship Project - Java Programming**

Console application that generates restaurant bills with 18% GST calculation for Syntecxhub.

### How to Run:
1. `javac RestaurantBilling.java`
2. `java RestaurantBilling`

### Features:
- Dynamic menu: Pizza Rs.250, Burger Rs.120, Coke Rs.60
- Automatic GST 18% calculation  
- Itemized billing with subtotal
- Input validation
